# 包公故里文化园信息展示网站（完整内容版）

## 打开方式

1. 解压压缩包。
2. 双击 `index.html` 即可浏览。
3. 电脑端和手机端均可使用。

## 已完成内容

- 首页景区总览、核心景观和文化主题
- 景区概览、官方导览图与推荐游线
- 包公故居、孝肃阁、包公书院、廉苑与文化展陈详情页
- 包拯生平、家训、精神品质及史实与传说辨析
- 航拍、建筑、园林和展陈分类图库
- 景区地址、公交、分季节开放时间、预约方式和咨询电话
- 官方联系宣传页和预约流程图，支持点击放大
- VR全景链接：https://www.720yun.com/vr/73djO5hvvf3
- 资料来源页面

## 主要文件

- `index.html`：首页
- `overview.html`：景区概览与导览图
- `landscapes.html`：景观导览
- `culture.html`：包公文化
- `gallery.html`：影像园区
- `service.html`：游客服务
- `sources.html`：资料来源
- `residence.html`：包公故居详情
- `xiaosu.html`：孝肃阁详情
- `academy.html`：包公书院详情
- `exhibition.html`：廉苑与文化展陈详情

## 信息提醒

开放时间、预约规则、公交线路及联系方式可能调整。正式发布或出行前，请通过“包游肥东”小程序或景区电话核实最新信息。

## 全站图片查看功能

- 正文内所有图片均可打开大图
- 电脑端支持单击、双击和键盘 Enter
- 手机端轻点图片即可打开
- 点击弹窗空白处、关闭按钮或按 Esc 可关闭
